# rmoid

Random [MongoDB ObjectId](https://docs.mongodb.com/manual/reference/method/ObjectId/) generator.

## Install

```
npm install rmoid
```

## Usage

```js
const rmoid = require("rmoid");

const randomObjectId = rmoid();
```

## CLI

```
npx rmoid
```
